House Prices In Southwark

I completed a challenging project for a client, where I was tasked with scraping real estate data from the Zillow website. Zillow has strict anti-scraping measures, but I utilized custom headers to bypass these restrictions effectively. Key Achievements: Data Extraction: I located the essential data embedded within a script tag on the webpage. By parsing this tag as JSON, I could extract detailed property information. Comprehensive Data Capture: The scraped data included property addresses, types, prices, bedroom and bathroom counts, square footage, broker names, and detailed URLs. Efficient Pagination: I implemented a solution to navigate through multiple pages, ensuring a comprehensive data collection covering all available listings. Technical Stack: Tools Used: Scrapy, Python, JSON Key Techniques: Custom headers for anti-scraping measures, JSON parsing, multi-page scraping This project showcases my ability to handle complex scraping tasks, overcome site restrictions, and deliver high-quality, structured data for real estate analysis.
